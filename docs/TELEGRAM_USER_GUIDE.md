# 텔레그램 연동 사용 가이드 (사용자용)

## 🎯 텔레그램 연동이란?

텔레그램 메신저로 실시간 알림을 받을 수 있는 기능입니다.

### 받을 수 있는 알림
- 🆕 신규 JD 등록 알림
- 📊 채용 프로세스 단계 변경 알림
- 🌅 매일 아침 9시 모닝 브리핑
- 📌 중요한 업데이트

---

## 🚀 10초 안에 연동하기

### 1단계: 설정 페이지 이동

```
https://jobizic-biz.vercel.app/settings
```

또는 상단 메뉴에서 **"⚙️ 설정"** 클릭

---

### 2단계: "텔레그램 연동하기" 버튼 클릭

- **"🚀 텔레그램 연동하기"** 버튼 클릭
- 텔레그램 앱이 자동으로 열립니다
- 6자리 코드가 표시됩니다

---

### 3단계: 자동 연동 완료!

- 텔레그램 앱에서 `/start` 명령어가 자동 실행
- ✅ **"연동이 완료되었습니다!"** 메시지 표시
- 끝! 이제 알림을 받을 수 있습니다 🎉

---

## 📱 텔레그램 봇 명령어

봇과 대화하면서 다음 명령어를 사용할 수 있습니다:

### `/today` - 오늘 할 일
```
📋 오늘의 업무
━━━━━━━━━━━━━━━

🆕 신규 JD: 3개
📊 활성 파이프라인: 12개
```

### `/pipeline` - 채용 프로세스 현황
```
📊 채용 프로세스 현황
━━━━━━━━━━━━━━━

서류전형: 5명
1차면접: 3명
2차면접: 2명
최종합격: 1명
```

### `/help` - 도움말
모든 명령어 목록을 볼 수 있습니다.

## 🔧 문제 해결

### Q: 텔레그램 앱이 자동으로 열리지 않아요
**A:** 아래 링크를 직접 클릭하세요
- 연동 화면에 표시된 "📲 텔레그램에서 열기" 링크 클릭

### Q: 코드가 만료되었어요
**A:** 코드는 5분간 유효합니다
- "취소" 버튼 클릭
- 다시 "텔레그램 연동하기" 버튼 클릭

### Q: 연동을 해제하고 싶어요
**A:** 설정 페이지에서 해제 가능합니다 (예정)
- 현재는 관리자에게 문의해주세요

### Q: 알림이 오지 않아요
**A:** 다음을 확인하세요
1. 텔레그램 연동 상태 확인 (설정 페이지)
2. 텔레그램 앱 알림 설정 확인
3. 봇을 차단하지 않았는지 확인

## 💡 팁

### 알림 받는 타이밍
- **즉시 알림**: 신규 JD 등록, 프로세스 단계 변경
- **정기 알림**: 매일 오전 9시 모닝 브리핑

### 봇과 대화하기
- 언제든지 봇에게 `/today` 입력하면 오늘 할 일 확인 가능
- `/pipeline`로 채용 현황 확인
