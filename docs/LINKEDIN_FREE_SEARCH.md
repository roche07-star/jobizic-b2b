# LinkedIn 무료 검색 가이드

**API 키 불필요!** 키워드를 추출해서 LinkedIn 검색 URL을 생성합니다.

---

## 🎯 작동 방식

1. **JD 선택** → Claude가 키워드 자동 추출
2. **검색 옵션 생성** → 여러 조합의 LinkedIn 검색 URL 생성
3. **클릭해서 검색** → LinkedIn 사이트에서 직접 검색
4. **후보자 탐색** → LinkedIn에서 프로필 확인

---

## ✅ 장점

- ✅ **완전 무료** - API 키 불필요
- ✅ **설정 없음** - 바로 사용 가능
- ✅ **무제한** - 검색 횟수 제한 없음
- ✅ **정확성** - LinkedIn 네이티브 검색 사용

---

## 📋 사용 방법

### 1. JD 선택

1. http://localhost:3000/recommendations 접속
2. **Searching(Linkedin)** 탭 클릭
3. 드롭다운에서 검색할 **JD 선택**

### 2. 무료 검색 실행

1. **✨ 무료 검색** 버튼 클릭
2. Claude가 키워드 자동 추출
3. 5개의 검색 옵션 생성:
   - **전체 검색** (추천) - 모든 키워드 조합
   - **직무 + 위치** - 핵심 직무와 지역
   - **전문 분야** - 수식어 + 직무
   - **핵심 스킬** - 주요 기술 스택
   - **포괄 검색** - 직무명만

### 3. LinkedIn에서 검색

1. 검색 옵션 **클릭**
2. LinkedIn이 새 탭에서 열림
3. **로그인** (LinkedIn 계정 필요)
4. 검색 결과 확인

### 4. 후보자 탐색

1. 프로필 클릭해서 상세 정보 확인
2. 적합한 후보자 **연결 요청** 또는 **InMail** 발송
3. 후보자 정보를 Jobizic에 수동으로 추가

---

## 💡 팁

### 검색 옵션 선택

- **"전체 검색 (추천)"** - 가장 정확한 결과
- **"직무 + 위치"** - 지역 한정 검색
- **"전문 분야"** - 특정 전문성 강조
- **"핵심 스킬"** - 기술 중심 검색
- **"포괄 검색"** - 넓은 범위 탐색

### 더 나은 결과를 위해

1. **명확한 직무명** - JD의 position 필드를 구체적으로 작성
2. **여러 옵션 시도** - 각 검색 옵션마다 다른 결과
3. **키워드 확인** - 자동 추출된 키워드가 적절한지 확인
4. **LinkedIn 필터** - LinkedIn 내부 필터로 추가 정제

## 🔍 예시

### JD 직무명
```
클라우드 SA (구축/운영/컨설팅)
```

### 자동 추출 키워드
- **핵심 직무**: 엔지니어
- **수식어**: 클라우드, 구축, 운영, 컨설팅
- **스킬**: 클라우드 아키텍처, AWS, Azure, GCP, DevOps

### 생성된 검색 옵션
1. **전체 검색**: `엔지니어 클라우드 구축 운영 클라우드 아키텍처 AWS Azure`
2. **직무 + 위치**: `엔지니어 한국`
3. **전문 분야**: `클라우드 구축 엔지니어`
4. **핵심 스킬**: `클라우드 아키텍처 AWS Azure`
5. **포괄 검색**: `엔지니어`

## ⚠️ 제한사항

### LinkedIn 로그인 필요
- LinkedIn 계정이 없으면 검색 불가
- 무료 계정으로도 가능

### 수동 작업 필요
- 자동으로 프로필을 가져오지 않음
- 사용자가 직접 LinkedIn에서 확인
- 적합한 후보자를 Jobizic에 수동으로 추가

### 검색 횟수 제한
- LinkedIn의 일일 검색 횟수 제한이 있을 수 있음
- 유료 LinkedIn Premium으로 제한 해제

---

## 🆚 유료 검색 vs 무료 검색

### 유료 검색 (Google Custom Search API)
- ✅ 자동으로 결과 표시
- ✅ Jobizic 내에서 확인
- ❌ API 키 필요 (설정 복잡)
- ❌ 100회/일 제한 (무료 할당량)
- ❌ 현재 403 에러로 작동 안 함

### 무료 검색 (LinkedIn 직접) ⭐
- ✅ API 키 불필요
- ✅ 설정 없이 바로 사용
- ✅ 무제한 검색
- ✅ LinkedIn 네이티브 검색 (더 정확)
- ❌ 수동 작업 필요
- ❌ LinkedIn 로그인 필요
